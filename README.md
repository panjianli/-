#(d)1.什么是L-属性文法？
一个属性文法称为L-属性文法，如果对于每个产生式A→X1X2…Xn，其每个语义规则中的每个属性或者是综合属性，或者是Xi(1in)的一个继承属性且这个继承属性仅依赖于：
(1) 产生式中Xi左边符号X1，X2，…，Xi-1的属性
(2) A的继承属性
S-属性文法一定是L-属性文法
